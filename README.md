# DataVis

Python projekt pro získávání dat z webových stránek a jejich analýzu.

## Funkce
- Web scraping pomocí `requests` a `BeautifulSoup`
- Analýza dat s `pandas`

## Instalace
1. Vytvořte si virtuální prostředí (doporučeno):
   ```powershell
   python -m venv venv
   .\venv\Scripts\activate
   ```
2. Nainstalujte závislosti:
   ```powershell
   pip install -r requirements.txt
   ```

## Použití
- Své skripty pro scraping a analýzu umístěte do složky `src/`.
- Spusťte hlavní skript:
   ```powershell
   python src/main.py
   ```

## Požadavky
Viz `requirements.txt` pro seznam závislostí.
